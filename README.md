# SENSESTICK Website

Freelance project under **ByteToBrain**. Production WordPress website for SENSESTICK®
— direct-to-spreadsheet measurement devices (TS Series temperature sensor family).

- **Client product**: Spreadsheet Measurement™ devices
- **Stack**: WordPress + Hello Elementor child + Elementor Pro + ACF Free + CPT UI + Calendly embed
- **Brand**: Primary `#000064` (navy), Secondary `#FFFF00` (yellow, accent-on-navy only)
- **Templates**: 11 effective Elementor templates covering 8 user-visible page types
  (Home, About, Contact, Book a Demo, Industries, Getting Started, Product Family,
  Downloads, Resource Article, Resource Listing, Spreadsheet Integration)
- **Design authority**: High-fidelity Figma exports in `figma/*.png` (19 frames)

## Repository layout
